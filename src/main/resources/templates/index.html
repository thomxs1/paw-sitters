Die Plattform, die Tierhalter und Gastgeber verbindet.

Typischer Ablauf:

  1. Tierhalter legt ein Profil an
  2. Tierhalter registriert ein Haustier
  3. Tierhalter erstellt eine Betreuungsanfrage fuer einen Zeitraum
  4. Gastgeber sendet ein Angebot
  5. Tierhalter nimmt ein Angebot an — andere werden automatisch abgelehnt
